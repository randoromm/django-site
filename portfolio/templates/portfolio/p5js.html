Snake clone (p5 JavaScript)

Click on the link above to play

This is a simple clone to a Snake game. I wrote the game using p5.js, which is a JavaScript library. P5.js has a full set of simplified drawing functionality which makes quick graphical experiments, like this one, easier to code.

To play the game, click on the link above and use either arrow keys or "WASD" keys. The game is not totally complete yet, but the fundamentals are there. Please keep in mind, that the canvas for the game only loads if your viewport(browser) width is more than 800 pixels, because this website is designed to be fully responsive on all viewport sizes and the canvas is 500x500px wide.

Mathematical Rose Patterns (p5 JS)

Click on the link above to view

In mathematics, a rose or rhodonea curve is a sinusoid plotted in polar coordinates. All these curves can be expressed by a polar equation of the form:

r = cos(k * θ)
or, alternatively, as Cartesian parametric equations:
x = cos(k * θ) * cos(θ)
y = cos(k * θ) * sin(θ)

This is a simple emulator of that with a few sliders to play around with different variables of the equation. You can read more about the rose on Wikipedia.org.
